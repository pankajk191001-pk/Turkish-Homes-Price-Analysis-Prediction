# Turkish Home Price Prediction

## Project Overview

This project predicts house prices in Turkey using machine learning
techniques and real estate property features.

## Business Problem

Real estate prices vary significantly depending on:

- Location
- Number of rooms
- Size (m²)
- Building age
- Floor number

The goal is to build a predictive model that estimates property prices accurately.

## Dataset Features

- Price
- Area (m²)
- Number of Rooms
- Number of Bathrooms
- Building Age
- Floor
- District
- City

## Machine Learning Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction

## Models Used

- Linear Regression
- Random Forest
- XGBoost (if used)

## Evaluation Metrics

- MAE
- RMSE
- R² Score

Best Model Accuracy: 99%
